# Project Setup

```sh
npm install
```

### Format your code before commit

```sh
npm run format
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Environment:

#### Development

```sh
npm run build-development
```

#### Production

```sh
npm run build-production
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
